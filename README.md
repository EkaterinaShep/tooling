## Режим с замедлением CPU 4x slowdown и эмуляцией сети Slow 3G

### Network

- [Профиль загрузки ресурсов при открытии страницы](/assets/profiles/slow.www.samsung.com.har)
- Неоптимальные места:
  - Дублирование ресурсов:
    - Приходит одинаковый ответ
      ![](/assets/images/2022-07-24_15-36.png)
  - Лишний размер ресурса. Некоторые файлы не сжимаются, минифицируются, оптимизируются. Несколько примеров:
    - js-файлы
      ![](/assets/images/2022-07-24_15-43.png)
    - Формат изображение jpeg вместо webp
      ![](/assets/images/2022-07-24_15-42.png)
  - медленно загружающиеся ресурсы. Несколько примеров:
    - больше 10 секунд загружаются SVG-спрайты, шрифты, некоторые js-файлы, видео и картинки
      ![](/assets/images/2022-07-24_15-46.png)
  - Ресурсы, блокирующие загрузку:
    ![](/assets/images/2022-07-24_15-53.png)
  - Некоторые файлы (.js, .css) не объединены в один файл (бандл),не используются полностью (удалить неиспользуемую часть) и/или медленно загружаются (удалить или заменить).
